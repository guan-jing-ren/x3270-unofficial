Known Bugs in x3270, c3270, wc3270, s3270, tcl3270, pr3287 and wpr3287 3.3.7

Bugs Fixed in the Current Patch Release

  • [x3270 3.3.7p7] Input of 8-bit characters (e.g., accented characters) does not work in x3270 in UTF-8 locales. (See
    • Gentoo bug 227997.)
  • [wc3270 3.3.7p7] Binary file transfer on Windows can result in corrupted files, with hex 0x0d characters inserted or deleted.
  • [wc3270 3.3.7p2] wc3270 could not start wpr3287.
  • [c3270 3.3.7p1] c3270's configure script would not detect missing ncurses header files, and c3270 would not build if ncursesw was not installed.

Outstanding Bugs

  • The x3270 pop-up keypad does not work properly with the Compiz Cube.
  • The Keep-alive feature does not work on Windows.
  • 'Flipped' mode (for right-to-left languages) is disabled.
  • File transfer does not support DBCS.
If you discover any other problems in this software, please contact Paul Mattes.
