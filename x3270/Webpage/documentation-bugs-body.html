Outstanding Bugs

  • The wc3270 manual page uses the wrong names for the resources used to set the font and font size for the PrintText action. The proper names are wc3270.printTextFont and wc3270.printTextSize respectively.
  • wc3270 default keyboard mappings that use the Alt key and a printable character, such as Alt-p for PrintText and Alt-c for Clear, do not always work. In particular, they will not be recognized if the Shift key is pressed or CapsLock is enabled.
  • s3270, tcl3270 and c3270 will not build with DBCS support turned off. The compile will fail at line 788 of glue.c. The workaround is to delete that line of code. Note that DBCS support is automatcially turned off in c3270's configure script if it cannot find a wide-curses library.
  • x3270 will not run correctly on a clean installation of Ubuntu 8.10. It will complain about not being able to find any colors and run in a white-on-black mode. This is not an x3270 bug, it is an Ubuntu bug. A workaround is to install your own copy of /etc/X11/rgb.txt. A copy is here.
  • The ibm_hosts file is ignored by x3270 and c3270.
  • When wc3270 copies DBCS text to the Windows clipboard, it adds an extra space character between each pair of DBCS characters.
  • 'Flipped' mode (for right-to-left languages) is disabled.
If you discover any other problems in this software, please contact Paul Mattes.
