Known Bugs in x3270, c3270, wc3270, s3270, tcl3270, pr3287 and wpr3287 3.3.8

Bugs Fixed in the Current Patch Release

  • The Ctrl-] mapping for the Escape action was inadvertently removed from wc3270.
  • On platforms that require the iconv library, c3270 would not recognize ncurses names in keymaps.
  • wc3270 would not start on Windows Vista.
  • The x3270 module keysym2ucs.c would not build on FreeBSD.
  • x3270 selections did not work properly on platforms that did not support XA_UTF8_STRING.

Bugs Fixed but not yet in a Patch Release

  • (none)

Outstanding Bugs

  • 'Flipped' mode (for right-to-left languages) is disabled.
